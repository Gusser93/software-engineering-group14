\subsubsection{Space Requirements}
%Keine?
\NFR{Die Datenmenge, die ein Gast auf den Server hochladen können darf, soll vom \gls{admin} beschränkt werden können.}{A}

\NFR{Automatische Backups sollen täglich, wöchentlich oder monatlich durch den \gls{admin} konfiguriert werden können.}{A}

\NFR{Der \gls{admin} soll alle Daten vor einem einstellbaren Datum löschen können.}{A}

\NFR{Für den Fall das die Speicherkapazität des \gls{server} überschritten wird, soll eine Fehlermeldung ausgegeben werden.}{A}

\NFR{Die Daten von nicht registrierten Nutzern werden, nach einer von dem \gls{admin} konfigurierbaren Zeit, automatisch gelöscht.}{A}

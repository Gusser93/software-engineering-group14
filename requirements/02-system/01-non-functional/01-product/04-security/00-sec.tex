\subsection{Security Requirements}
\NFR{Sämtlicher Datenaustausch zwischen \gls{client} und \gls{server} soll über eine verschlüsselte \gls{ssl} Verbindung erfolgen.}{A}

% Nachfragen
\NFR{Alle hochgeladenen Daten sollen nicht verschlüsselt in einer Datenbank auf dem Server abgespeichert werden.}{A}

\NFR{Für die Implementierung von Verschlüsslungsalgorithmen soll auf \gls{opensource} Projekte zurückgegriffen werden, wie z.B. \gls{openssl}.}{A} 

% Nachfragen
\NFR{Benutzerpasswörter müssen mindestens 8 Zeichen lang sein und aus Groß-, Kleinbuchstaben, sowie Zahlen bestehen.}{A}

% Nachfragen
\NFR{Benutzerpasswörter werden verschlüsselt auf dem Server gespeichert.}

% Nachfragen
\NFR{Die Zugriffsrechte von \glspl{Plugin} sollen über eine \gls{sandbox} beschränkt werden.}{A}

\subsection{Security Requirements}
\NFR{Data transfer to and from the server must be
    performed via an \gls{HTTPS} connection.}{A}

\NFR{Peer-to-Peer communication shall be encrypted with
    a hybrid encryption algorithm.}{A}

\NFR{The invitation URL for guests shall only be usable once.}{A}

\NFR{Passwords are stored on the server in text/plain,
    but hashed with a random seed.}{A}

\NFR{All data (shared data and session recordings) is
    stored unencrypted on the server.}{A}

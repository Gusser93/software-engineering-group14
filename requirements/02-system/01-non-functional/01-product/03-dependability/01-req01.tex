\NFR{If the system crashes, the system shall try to reboot
    automatically and inform the administrator.}{A}

\NFR{Backups of stored data (session recordings, shared
    files, user accounts) shall be done daily and stored on a different
    server.}{A}

\NFR{The system shall be available 24/7.}{A}

\NFR{The number of system crashes in a mounth shall
    be less than one.}{A}

\NFR{There is no need to stop the application to perform
    a backup.}{A}
